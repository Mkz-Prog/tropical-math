# tropical-math

Репозиторий посвящён разделу математики под названием тропическая математика. Пакет представляет из себя базовую функциональность действий в тропическом полукольце.

# Фунциональность

В пакете предоставлены тропическое сложение и тропическое умножение в мин-алгебре.

Само число -- любое число, элемент множества R U {+inf}.

```go
import "github.com/Mkz-Prog/tropical-math/math"

r :=  math.Rational{Value: v, IsInfinity: false}
```

## Тропическое сложение

Для сложения чисел имеется метод Add:

```go
result = r1.Add(r2)
```

## Тропическое умножение

Для сложения чисел имеется метод Multiply:

```go
result = r1.Multiply(r2)
```
